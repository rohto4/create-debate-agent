# AGENTS.md Template Guide

`AGENTS.md` は、agentが最初に読む最上位ルールとして扱う。

実体化には `templates/AGENTS.md` を使う。

## 必ず含める内容

- 日本語・UTF-8で進めること
- secretを書かないこと
- PJ固有の目的と扱う情報
- 読み込み順
- docs配下の情報の置き場所
- 回答方針
- ECCの扱い

## 置換値

- `<PJ-NAME>`: PJ名
- `<PJ-SLUG>`: slug
- `<PJ-PURPOSE>`: PJ目的
- `<PJ-POLICY>`: PJ固有方針
- `<PJ-FEATURES>`: 主な成果物や機能
