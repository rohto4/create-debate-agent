# Docs Structure

標準の `docs/` 構成。

```text
docs/
  README.md
  ecc-<PJ-SLUG>-setup.md
  memo.md
  candi-ref/
    README.md
    candi-ref-summary.md
  diary/
  guide/
    README.md
    guide-summary.md
  imp/
    README.md
    imp-plan.md
    imp-comp.md
    imp-wait.md
    user-tasks.md
  spec/
    README.md
    spec-summary.md
  setting/
    README.md
    init.md
    templates/
```

## 役割

- `guide/`: 採用済みの運用ガイド、判断基準、ルール補足
- `spec/`: 確定した仕様、要件、設計前提
- `candi-ref/`: 候補案、調査、比較、未採用案
- `imp/`: 作業計画、実装メモ、完了記録、ユーザー作業
- `diary/`: セッション単位の作業記録
- `setting/`: 初期化用テンプレート、設定資料
- `memo.md`: 一時メモ。確定したら適切な場所へ移す。
