# 代替プラットフォーム調査

作成日: 2026-05-15

## 概要

Slack / Discord 以外でマルチエージェント討論を動作させることができるプラットフォームを調査した。

---

## 1. エージェントネイティブプラットフォーム

### 1.1 MoChat

- **URL**: https://mochat.io/
- **特徴**: エージェント通信のためにネイティブに構築されたプラットフォーム
- **機能**: チャンネル、グループ作成、プラグインエコシステム（MCP対応）
- **利点**: エージェントがファーストクラス市民。Slack/Discord の制約なし
- **欠点**: エコシステムがまだ初期段階

### 1.2 FloatIM

- **URL**: https://floatboat.ai/floatim
- **特徴**: エージェントネイティブメッセージングネットワーク
- **機能**: 人間とエージェントが同じグループチャット。IACT / Selfware プロトコル
- **利点**: エージェントが第一級市民。ローカル実行対応
- **欠点**: 利用実績がまだ少ない

### 1.3 Agent United

- **URL**: https://agentunited.ai/
- **特徴**: 全エージェントを1つのチャットに集約
- **機能**: DM、グループチャット、共有チャンネル。永続履歴
- **利点**: 自分でホスト可能。OpenClaw、AutoGPT、CrewAI 等と互換
- **欠点**: 有料プラン必要な機能あり

---

## 2. チームチャットプラットフォーム

### 2.1 Microsoft Teams

- **特徴**: 企業向けチームチャット。Bot Framework 対応
- **エージェント対応**: Teams Bot SDK、Azure Bot Service
- **利点**: 企業利用が多い。Office 365 との統合
- **欠点**: 開発者体験が Slack より劣る

### 2.2 Google Chat

- **特徴**: Google Workspace 内のチャット
- **エージェント対応**: Google Chat API、Apps Script
- **利点**: Google Workspace 利用者向け
- **欠点**: エコシステムが限定的

### 2.3 Telegram

- **特徴**: オープンなメッセージングプラットフォーム
- **エージェント対応**: Telegram Bot API
- **利点**: API がシンプル。グローバルに利用可能
- **欠点**: 企業利用には不向き

---

## 3. オープンソースプラットフォーム

### 3.1 Hazel

- **URL**: https://hazel.sh/
- **特徴**: オープンソースのチームチャット。Slack 代替
- **機能**: リアルタイムメッセージング、GitHub / Linear 統合、AI エージェント対応
- **利点**: Discord / Slack / Hazel 間でメッセージ同期可能
- **欠点**: まだ発展途上

### 3.2 CoChat AI

- **URL**: https://cochat.ai/
- **特徴**: Open WebUI ベースの AI ワークスペース
- **機能**: チーム共有 AI 会話、サンドボックス実行、70+ 統合
- **利点**: 自分でホスト可能。プライバシー重視
- **欠点**: マルチエージェント協調の仕組みは自前で構築が必要

### 3.3 Mantle Chat

- **URL**: https://mantle.chat/
- **特徴**: マルチモデル対応の AI チャットプラットフォーム
- **機能**: Claude、GPT、Gemini、Grok、DeepSeek を1つのサブスクリプションで
- **利点**: モデル切替が容易。チーム共有対応
- **欠点**: エージェント協調機能は限定的

## 5. 推奨

### 初期（MoChat）

- エージェントネイティブ性を重視し、MoChat API を直接呼び出す方式で開始
- Slack は既存ワークスペース連携が必要になった場合の代替案として保持

### 将来（追加検討）

- FloatIM や Agent United など、より強い自ホスト要件やエージェント間通信要件が出た場合に再比較する
- Slack/Discord は利用者の既存ワークフローに寄せる必要がある場合に検討する
