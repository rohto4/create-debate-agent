# PROJECT.md Template Guide

`PROJECT.md` は、PJ固有の目的、構成、運用方針をまとめる正本として扱う。

実体化には `templates/PROJECT.md` を使う。

## 必ず含める内容

- PJの目的
- PJ固有方針
- 主な成果物
- 優先度モデル
- ECCコピー元
- 取り込み対象skill / command
- docs運用ルール

## 置換値

- `<PJ-NAME>`: PJ名
- `<PJ-SLUG>`: slug
- `<PJ-PURPOSE>`: PJ目的
- `<PJ-POLICY>`: PJ固有方針
- `<PJ-FEATURES>`: 主な成果物や機能
- `<ECC-ROOT>`: ECC clone path
- `<ECC-EXPAND-ROOT>`: ecc-expand clone path
